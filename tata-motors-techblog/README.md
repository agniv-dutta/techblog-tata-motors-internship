## Installation

1. Navigate to the project directory:
```bash
cd tata-motors-techblog
```

2. Install dependencies:
```bash
npm install
```

## Running the Project

Start the development server:
```bash
npm start
```

The app will open at [http://localhost:3000](http://localhost:3000)

## Building for Production

```bash
npm build
```
